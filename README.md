«Brain Games» – это набор из пяти консольных игр, построенных на взаимодействии с пользователем. Каждая игра задаёт вопросы, на которые нужно дать правильные ответы. После трёх правильных ответов игра считается завершённой. Неправильный ответ завершает игру и предлагает пройти её заново.

## Доступные игры

1. **brain-even** – Определение чётного числа
2. **brain-calc** – Вычисление арифметических выражений
3. **brain-gcd** – Нахождение наибольшего общего делителя
4. **brain-progression** – Поиск пропущенного числа в последовательности
5. **brain-prime** – Определение простого числа

## Системные требования

- Node.js версии 13 и выше
- npm (обычно устанавливается вместе с Node.js)

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/hodoknow/fullstack-javascript-project-44.git
   ```

2. Перейдите в директорию проекта:
    ```bash
    cd fullstack-javascript-project-44
    ```

3. Установите зависимости:
    ```bash
    npm install
    ```

4. Установите пакет глобально (для доступа к играм из любой директории):
    ```bash
    npm link
    ```

## Как играть

После установки вы можете запустить любую из игр, используя соответствующие команды:
* brain-even
* brain-calc
* brain-gcd
* brain-progression
* brain-prime
